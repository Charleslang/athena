# const  

使用 `const` 来声明常量。

如果使用 `const` 声明一个普通类型的变量，则该变量必须在声明时赋值，并且不能再被重新赋值。

```js
const a = 1;
const c; // 报错
```

如果使用 `const` 声明一个引用类型的变量，则该变量的引用地址不能被修改，但可以修改其属性。

```js
const c = { }
c.name = 'hello'; // 可以进行此操作
c = { } //非法操作 (c 的地址被修改) 
```

注意：

- `const` 声明的常量也具有块级作用域。
- `const` 声明的常量不存在变量提升。
